# `test`

This folder contains project tests and exploratory validation scripts.

## Current focus

- `visualizaciones/`: plotting scripts used to inspect hydraulic laws,
  compare models, and evaluate approximation errors.

At the moment the emphasis is on visual and numerical validation rather
than on a large automated unit-test suite.

This means the folder is currently used more as an engineering notebook
with executable scripts than as a strict `pytest`-style test suite.
