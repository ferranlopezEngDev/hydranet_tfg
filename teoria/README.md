# `teoria`

This folder stores the theoretical references that guide the
implementation.

## Current contents

The PDFs in this directory cover the TFG formulation and supporting
fluid-mechanics material used while deriving or checking the current
models.

These documents are reference material for humans. They are not imported
or processed by the Python code.

## Expected workflow

1. Start here when the intended equation, notation, or sign convention
   is unclear.
2. When the code makes a numerical simplification or continuity choice
   that is not literally the textbook formula, keep that choice
   documented in code and traceable back to the theory.
3. Treat this folder as supporting documentation, not as executable
   input data.
