# `src/hydraulic_solver/H_based_elements/nodes`

This folder defines node-side abstractions for the H-based formulation.

## Current status

- `NODES.py` contains the abstract interface expected from hydraulic
  nodes.

This area is still much less developed than the pipe models and will
likely grow when network assembly is added.

At the moment the file acts mainly as a design placeholder so the node
side of the formulation is documented even before full implementations
are added.
